Editing Time Tables

Best viewed in Firefox, Chrome, etc.

Creating / Deleting Time Tables
Click the button to create a new table. When the dialog opens give it a name. Note: A table must have a name before it can be created.

IMPORTANT! Multiple tables can be created, however, a table must have focus before lines can be added to it or deleted from it.

The default name for new tables will be in one of the following formats:

  1. Long name: Wed Nov 21 2018 (default)
  2. Short name: 11/21/2018

Double click the table name to open a dialog box that allows user input.
Click the checkmark in the table header to delete the table.

Adding / Deleting Time Lines
Click the button to create a new line in the table that has focus. The new line will contain the current time in the "Start" cell. The "Stop" cell will contain the current time +1 hour.

Click the checkmark next to any row or table to delete it.

Inserting Time Lines
Double click any "Hrs" cell to insert a new row below the row of the clicked cell.

Editing Time Lines
Double click a "Description" cell to open a dialog box that allows user input.

Double click a "Start" cell to insert the current values of both time fields into both the "Start" & "Stop" cells.

Double click a "Stop" cell to insert the current value of the "Stop" time field.

Click the envelope in the bottom left of the footer to email the author.
