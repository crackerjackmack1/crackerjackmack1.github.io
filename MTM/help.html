Editing Time Tables

Best viewed in Firefox, Chrome, etc.

Creating & Deleting Time Tables

To Create a Time Table...

  1. Under "Table Name" choose the date style that will appear as the table's name.
    • Long date = Wed Nov 21 2018 (default)
    • Short date = 11/21/2018
  2. Click the Table button to continue.
  3. A dialog box will open allowing the user to modify the name of the table.
  4. Click "OK" to create the table.
  5. Note: Tables can not be created without a name, but table names can be edited after creation!

To Modify a Table's Name after Creation...

  1. Double click the table's "Name" header.
  2. A dialog box will open to allow user to modify the table's name.
  3. Click "OK" to continue.

To Remove a Time Table...

  1. Click the ✖ in the upper right corner.

Adding, Inserting & Deleting Time Lines

IMPORTANT! All times in the "Start" & "Stop" columns of a table are displayed in a "24 hour" format. This is mandatory and cannot be edited!

To add a Time line at the top of a table...

  1. Click the ➕ symbol to the left of the "Description" column header.
  2. Modify as needed.

To insert a new time line under an existing line...

  1. Double click any "Hrs" cell to insert a new time line below it.
  2. Modify as needed.

To delete a Time line from a table...

  1. Click the checkmark (✓) to the left of any time line to delete it.

Note: All time lines added/inserted will display the current time in the "Start" cell, and the current time plus 2 minutes in the "Stop" cell (all in 24 hour format).

Editing Time Lines

IMPORTANT! A table must contain at least one timeline before the following actions can be applied!

Double click any cell under the "Description" header to open a dialog box that will allow user input.

Double click any cell under the "Start" header to insert the current values of the "Start Time" and "Stop Time" fields into their corresponding cells.

Double click any cell under the "Stop" header to insert the current value of the "Stop Time" field into it.
