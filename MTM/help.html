Editing Time Tables

Best viewed in Firefox, Chrome, etc.

Creating Time Tables

To create one or more tables on the page:
  1. Under "Date/Name" choose "Long" or "Short" for the date name that will appear in the table header.
  2. Under "Intervals" choose "5 min" or "6 min" for the length of interval to use in calculations for the table.
    Note: Each table's interval length is shown in botton left corner.
  3. Press the "Table" button to create a new table.
  4. Use either the default "Name" that appears in the dialog box or modify it as needed.

    The default name for new tables will be in one of the following formats:

    • Long date: Wed Nov 21 2018
    • Short date: 11/21/2018
    Tables will not be created without names.

  5. Click "OK"

Deleting Time Tables

To remove a table form the page "ClicK" the ✖ in the upper right corner of the table header.

Adding / Deleting Time Lines
Click the checkmark in the upper left of the table. The new Line will contain the current time in the "Start" cell, and the "Stop" cell will contain "Start" time +1 hour.

Click the checkmark next to any row to delete it.

Inserting Time Lines
Double click a "Hrs." cell to insert a new row under the clicked cell.

Editing Time Lines
Double click "Description" cells to open a dialog box that allows user input.

Double click a "Start" cell to insert the current values of both time fields into both the "Start" & "Stop" cells.

Double click a "Stop" cell to insert the current value of the "Stop" time field.

Click the envelope in the bottom left of the footer to email the author.
