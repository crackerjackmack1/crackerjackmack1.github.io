Editing Time Tables

Best viewed in Firefox, Chrome, etc.

Creating & Deleting Time Tables

To create one or more tables on the page:

  1. Under "Date/Name" choose "Long" or "Short" for the date name that will appear in the table header.
  2. Under "Intervals" choose "5 min" or "6 min" for the length of interval to use in calculations for the table.
    Note: Interval lengths are shown in botton left corner.
  3. Press the "Table" button to create a new table.
  4. Use either the default "Name" that appears in the dialog box or modify it as needed.

    The default name for new tables will be in one of the following formats:

    • Long date: Wed Nov 21 2018
    • Short date: 11/21/2018
    Note: Tables will not be created without names.

  5. Click "OK"

To remove a table from the page "Click" the ✖ in the upper right corner of the table header.

Adding & Deleting Time Lines

To add a line at the top of a table:

  1. Click the checkmark in the upper left of the table.
  2. (The new Line will contain the current time in the "Start" cell, and the "Stop" cell will contain "Start" time +2 minutes.)

To delete a line from a table:

  1. Click the checkmark next to any row to delete it.

Editing Time Lines

Double click any "Description" cell to open a dialog box that allows user input.

Double click any "Start" cell to insert the current values of the time fields into the "Start" & "Stop" cells.

Double click any "Stop" cell to insert the current value of the "Stop" time field into it.

Inserting Time Lines

Double click any "Hrs" cell to insert a new row below the level of the clicked cell.
