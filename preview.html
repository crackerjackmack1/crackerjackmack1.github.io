"My Time Machine"

Sep 2, 2017

Weight: 12.6 KB

Originally, this was a very basic time keeping sheet that could be printed, and then filled in by hand. This newer concept allows the user to create tables and add some rows with a few button clicks.

To edit a table's "Header" or "Descriptions" double click on the cell. To insert a row under an existing row, double click cells in the "Hrs" column.

Unchecking the column headers row removes the table (same for rows). To save: print the tables as a PDF (not available on all mobile devices).

0.1 hours = 5 minutes
Entries 2 minutes long or less = 0.0
